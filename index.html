GLIMMER

A Visual LVG analysis tool

View the Project on GitHub deanmark/glimmer

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

Large velocity gradient (LVG) analysis is the most widely used method of radiative transfer. GLIMMER is a MATLAB program that allows you to perform LVG computations and their analysis in a user friendly graphical interface.
